Formatting Guide

Use Markdown in the parse_mode field. Syntax:

*bold text*
_italic text_
[inline URL](http://www.example.com/)
`inline fixed-width code`
```pre-formatted
```python
Python code block
```

Note: No nested entities. Use MarkdownV2 for advanced formatting.

Use MarkdownV2 for advanced formatting. Syntax:

*bold \*text*
_italic \*text_
__underline__
~strikethrough~
||spoiler||
[inline URL](http://www.example.com/)

Escape special characters with \. Supports nested entities.

Use HTML for precise formatting. Supported tags:

<b>bold</b>
<i>italic</i>
<u>underline</u>
<s>strikethrough</s>
<a href="http://www.example.com/">inline URL</a>

Replace <, >, and & with HTML entities.
